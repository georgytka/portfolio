# Portfolio

Статический сайт-портфолио на HTML/CSS/JS.

## Локальный запуск

Самый простой способ:
- открыть папку проекта в VS Code
- поставить расширение Live Server
- запустить `index.html` через Live Server

## Деплой на GitHub Pages

1. Создай репозиторий на GitHub
2. Загрузи файлы проекта
3. Открой `Settings` → `Pages`
4. Выбери `Deploy from a branch`
5. Укажи ветку `main` и папку `/root`
6. Сохрани настройки

Сайт будет доступен по ссылке GitHub Pages.
